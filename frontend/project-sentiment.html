Real-Time Sentiment Analysis

Streaming NLP pipeline analysing live social media data using transformer models.

BERT Kafka Python Docker

Problem Statement

Businesses need immediate insight into public sentiment to react to product launches, outages, or brand crises in real time.

Architecture

  • Live data ingestion via Kafka
  • Text preprocessing and batching
  • BERT-based sentiment classification
  • Streaming output to dashboard and database

Results

  • Processed 50,000+ messages per day
  • Achieved 90%+ classification accuracy
  • Latency under 2 seconds end-to-end

Key Insights

  • Negative sentiment spikes strongly correlated with service outages
  • Neutral sentiment dominated outside peak news cycles
  • Positive sentiment increased after rapid response messaging
