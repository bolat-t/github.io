Customer Churn Prediction

End-to-end machine learning project predicting customer churn using structured business data and gradient boosting models.

Python XGBoost Pandas Scikit-learn

Problem Statement

Customer churn is a major driver of revenue loss for subscription-based businesses. The goal of this project was to identify customers at high risk of churning and provide interpretable insights that could support retention strategies.

Data Overview

  • Customer demographics
  • Account tenure and contract type
  • Service usage metrics
  • Billing and payment history

Approach

  • Exploratory Data Analysis (EDA)
  • Feature engineering and encoding
  • Class imbalance handling using SMOTE
  • XGBoost model training and tuning
  • Model evaluation with ROC-AUC and confusion matrix

Results

The final model achieved:

  • 92% accuracy
  • 0.94 ROC-AUC score
  • Strong recall on churned customers

Key Insights

  • Month-to-month contracts showed the highest churn rates
  • Short tenure customers were significantly more likely to churn
  • Electronic check payments correlated with higher churn
