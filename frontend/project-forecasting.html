Sales Forecasting Dashboard

Time-series forecasting system supporting data-driven business planning.

Prophet ARIMA SQL Plotly

Business Context

Accurate sales forecasting is critical for inventory planning, staffing, and revenue projections.

Live Power BI Dashboard

Interactive forecasting dashboard with multi-horizon predictions and confidence intervals.

Methodology

  • Historical sales aggregation using SQL
  • Seasonality and trend decomposition
  • Prophet and ARIMA model comparison
  • Forecast evaluation using MAPE

Results

  • 35% reduction in forecasting error
  • Improved quarterly planning accuracy
  • Interactive dashboard adoption by stakeholders

Impact

  • Reduced stock-outs during peak demand
  • Lower excess inventory costs
  • Data-driven decision-making across teams
